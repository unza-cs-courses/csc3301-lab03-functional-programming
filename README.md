# Lab 3: Functional Programming and Higher-Order Functions

**CSC3301 Programming Language Paradigms**  
**Points:** 100

## Tasks

### Task 1: Pure Function Transformation (20 points)
Convert impure functions to pure equivalents.

### Task 2: HOF Implementation (30 points)
Implement `my_map`, `my_filter`, `my_reduce`, `my_zip_with` WITHOUT using built-ins.

### Task 3: Composition and Pipelines (25 points)
Implement `compose()`, `pipe()`, and `curry()`.

### Task 4: Refactoring Challenge (25 points)
Rewrite imperative code using only map/filter/reduce (NO LOOPS).
